## Workflow on the day (45 min before first race)

1. **Open carnivaltiming.com on the timing phone**
2. Tap **Join Existing Carnival** (NOT new — Paddy created it earlier this week)
3. Type `WD26` (capitals, no dash) → tap Join
4. App asks for access code → type `WPS-2026` (capitals, dash before 2026) → Validate
5. App will show "Race Control / Marshal / Lane Timer" choice → tap **Marshal** (XC mode)
6. App will show the active race (e.g. "9 Girls"). If wrong race shown, tap the dropdown to pick the right one.

## During each race

1. **Pin bibs** on runners 5 minutes before each race. WPS bibs are colour-coded green.
2. As runners cross the finish line, marshal **types the bib number** into the app (or taps Pick → name list if bib is unreadable).
3. Each finisher's name + time appears on the live page **within 3 seconds**.
4. After the race ends, tap **Publish Race** → results lock + appear in spectator view.

## If something goes wrong

| Problem | Fix |
|---|---|
| App says "Carnival not found" | Code typed wrong. Try `wd26` (lowercase) or copy from Paddy's note. |
| Access code rejected | Type exactly `WPS-2026` with the dash. If still rejected, call Paddy. |
| Phone battery dead | Plug in power bank. App auto-resumes. Don't close the tab. |
| 4G slow | Switch phone to airplane mode for 5 sec, then back on. |
| Wrong runner's name appearing | Tap the time → "Edit time" / pick correct runner from drop-down. |
| Need to undo a finisher | Tap their row → swipe left → Remove. (Admin mode only.) |
| Bib unreadable | Tap **📋 Pick** button (XC Marshal screen) → tap runner's name from filtered list. |
| Sacred Heart attends with extras | Use spare blank bibs + sharpie. Add to roster manually after the race. |
